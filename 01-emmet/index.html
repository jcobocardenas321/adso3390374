01-Emmet

Pluguin for fater HTML and CSS coding.

01-Child >

Creates child elements (nesting)

Example:

ul>li>a

Result:

<ul> <li><a href=""></a></li> </ul>

02-Sibling +

Creates sibling elements (horizontal)

Example:

section+header+nav

Result:

<section></section> <header></header> <nav></nav>

03-Multiplication *

Repeats elements multiple times

Example:

ul>li*3

Result:

<ul> <li></li> <li></li> <li></li> </ul>

04-Class .

Creates a class attribute

Example:

div.contenedor

.contenedor

Result:

<div class="contenedor"></div>

05-Classes ...

Creates multiple class attributes

Example:

div.clase1.clase2.clasen

Result:

<div class="clase1 clase2 clasen"></div>

06-Id #

Creates an id attribute

Example:

nav#menu

Result:

<nav id="menu"></nav>

07-Text {}

Inserts text content

Example:

div{Lorem Ipsum Dolor}

a{Click Aquí}

Result:

<div>Lorem Ipsum Dolor</div> <a href=“”>Click Aquí</a>

08-Custom Attribute []

Creates custom attributes

Example:

a[href=www.google.com]

Result:

<a href="www.google.com"></a>

09-Item Numbering $

Adds consecutive numbers

Example:

ul>li.num$*3

section>article.sec$$$*3

Result:

<ul> <li class="num1">1</li> <li class="num2">2</li> <li class="num3">3</li> </ul>

10-Reverse Numbering @-

Inverts numbering order

Example:

ul>li.num$@-*3

Result:

<ul> <li class="num3">3</li> <li class="num2">2</li> <li class="num1">1</li> </ul>

11-Base Numbering @n

Defines a base number for sequences.

Example:

ul>li.num$@3*4

Result:

<ul> <li class="num3"></li> <li class="num4"></li> <li class="num5"></li> <li class="num6"></li> </ul>

12-Climb Up^

Moves up one level.

Example:

section>article>div^p

Result:

<section> <article> <div></div> </article> <p></p> </section>

13-Grouping( )

Groups elements together.

Example:

(header>nav)(section>article)

Result:

<header> <nav></nav> </header> <section> <article></article> </section>

14-Loremlorem

Generates placeholder text.

Example:

lorem-lorem20-lorem*5

Result:

lorem + enter = lorem is for generating filler text

15-HTML Documents!

Creates various HTML document types.

Example:

html:xs Documento xhtml 1.0 Strict
html:xt —— Documento xhtml 1.0 Transitional
html:4s —— Documento html 4.01 Strict
html:4t —— Documento html 4.01 Transitional
html:5 —— Documento html 5

Result:

Example of html:5 -- ! + TAB <!DOCTYPE html> <html lang="en"> <meta charset="UTF-8"> <title>Document</title> <head> ... <body> ... </html>

16-Meta Tagsmeta:

Generates meta tags.

Example:

meta:vp

Result:

<meta name="viewport" content="width= device-width, initial-scale=1.0">

17-Challengechallenge:

Generates Emmet challenges for practice.

Example:

Realice Grouping, Climb Up and Base Numbering

Result:
